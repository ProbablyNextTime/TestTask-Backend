
## Start server scripts

### `npm build`
Builds the app to buildBack dir.

### `npm start`
Launches builded project from buildBack dir

## Dev scripts setup

### `npm run watchTS`
Autocompile changes.

### `npm run watchNode`
Autorestart server on changes.
